# 🎮 MyMetroidvania

2D-платформер в стиле метроидвания, разработанный на C# с использованием фреймворка MonoGame. Проект создан с акцентом на чистую архитектуру, паттерны проектирования и масштабируемость игровых систем, что делает его удобным для дальнейшего расширения.

## ✨ Ключевые особенности
- **Архитектура MVC:** Чёткое разделение игровой логики, обработки ввода и отрисовки. Каждая сущность (игрок, враг, чекпоинт) реализована как независимая тройка `Model-View-Controller`.
- **Паттерн Observer:** Событийная система для обновления UI и игровых состояний без жёстких связей между компонентами.
- **Продвинутая механика игрока:** Переменная высота прыжка, двойной прыжок, 4-направленная ближняя атака, система здоровья с кадрами неуязвимости и отбрасыванием при получении урона.
- **ИИ врагов:** Патрулирование, преследование при обнаружении игрока, обработка столкновений с платформами и реакцией на урон.
- **Динамическая камера:** Плавное следование за игроком с вертикальным смещением при прицеливании вверх/вниз.
- **Система чекпоинтов:** Точки сохранения с механизмом возрождения и визуальной индикацией активности.
- **Модульный интерфейс:** Полоска здоровья, отрисовываемая независимо от игрового мира и камеры.

## 🛠️ Технологии и стек
- **Язык:** C# (.NET 9)
- **Фреймворк:** MonoGame 3.8 (DesktopGL)
- **Архитектура:** MVC, Observer Pattern
- **Инструменты:** VS Code / Visual Studio, Git

## 🎮 Управление
| Клавиша | Действие |
|---------|----------|
| `A / D` или `← / →` | Движение |
| `Пробел` (тап/удержание) | Прыжок (переменная высота) |
| `Пробел × 2` | Двойной прыжок |
| `J` | Атака |
| `W/↑ + J` | Атака вверх |
| `S/↓ + J` | Атака вниз |
| `W/↑` или `S/↓` | Смещение камеры |
| `Esc` | Выход |

## 📚 Цель проекта
Разработка структурированного игрового прототипа с упором на чистый код, архитектурные паттерны и масштабируемость. Все системы реализованы с разделением ответственности, что упрощает юнит-тестирование, расширение механик и интеграцию новых функций без переписывания существующего кода.
