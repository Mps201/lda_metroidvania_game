# 🎮 MyMetroidvania

2D-платформер в стиле метроидвания, разработанный на C# с использованием фреймворка MonoGame. Проект создан с акцентом на чистую архитектуру, паттерны проектирования и масштабируемость игровых систем.

## ✨ Ключевые особенности

### 🏗️ Архитектура
- **MVC (Model-View-Controller):** Чёткое разделение данных, логики и отрисовки. Каждая сущность (игрок, враг) реализована как независимая тройка компонентов.
- **Observer Pattern:** Событийная система (`OnHealthChanged`, `OnDamageTaken`) для реактивного обновления UI без жёстких связей.
- **Game State Manager:** Управление глобальными состояниями приложения (Title → Playing → Paused) с изолированными циклами обновления.

### 🎮 Геймплей
- **Продвинутая механика игрока:** Переменная высота прыжка, двойной прыжок, 4-направленная атака, отбрасывание при уроне.
- **ИИ врагов:** Патрулирование платформ, преследование игрока при сближении, умный разворот у краёв.
- **Система здоровья:** 3 сердца, кадры неуязвимости с визуальным миганием, респаун на чекпоинтах.
- **Переходы между уровнями:** Зоны телепортации с загрузкой новых конфигураций уровня (тайлы, спавны, выходы).

### 🎨 Визуал и интерфейс
- **Динамическая камера:** Плавное следование за игроком (Lerp) с вертикальным смещением.
- **Модульный UI:** HealthBar, отрисовываемый независимо от камеры; титульный экран и меню паузы.
- **Отладочная визуализация:** Полупрозрачные зоны выхода, цветные хитбоксы атак.

## 🎮 Управление

| Клавиша | Действие |
|---------|----------|
| `A / D` или `← / →` | Движение влево/вправо |
| `Пробел` (тап) | Короткий прыжок |
| `Пробел` (удержание) | Высокий прыжок |
| `Пробел × 2` | Двойной прыжок |
| `J` | Атака мечом |
| `W/↑ + J` | Атака вверх |
| `S/↓ + J` | Атака вниз |
| `W/↑` или `S/↓` | Смещение камеры для обзора |
| `Esc` | Пауза / Возврат в игру |
| `Enter` / `Space` | Запуск игры с титульного экрана |


**Принцип работы:**
- **Title:** Ожидание ввода для старта, отрисовка заголовка.
- **Playing:** Полный игровой цикл (ввод → обновление моделей → коллизии → отрисовка).
- **Paused:** Заморозка обновления игровых объектов, отрисовка оверлея поверх мира.

### Требования
- [.NET 9 SDK](https://dotnet.microsoft.com/download)

cd lda_metroidvania_game
